### Hide a File/Folder

1. **Right-click** on any file or folder in the file explorer
2. Select **Hide**
3. The file disappears instantly (but still exists)

### Unhide a File

1. Navigate to where the hidden file was
2. **Right-click** on that location
3. Select **Unhide**
